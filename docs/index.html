// Local

whisper.cpp runs on your laptop. No API key. No network. ~250 ms transcription for a 5-second clip on M2 with the small model.

// Push-to-talk

Hold the hotkey, talk, release. No wake word, no five-second timeout that cuts you off mid-thought. Works anywhere cmd+V works.

// Voice → Claude Code

Turn on auto_paste + auto_submit in the config: focus Claude chat → hold ⌥ → speak → message lands and sends. No keyboard, no clicks.
